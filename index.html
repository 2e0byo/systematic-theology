Bibliography

For convenience the bibliography is available here. Do bear in mind the comments in the first lecture about how to use a bibliography effectively! Reading for each week will be posted via moodle and emailed around.

Note that many of the Internet Archive links are short-term loans, i.e. you can borrow them for an hour. This means only one person can read the book at once! Thus if the book is unavailable you have three options: 1. come back later, 2. search archive.org for another copy, 3. bribe/threaten/cajole/persaude whoever is reading it to let you have it. If you go for 3. there's a fair chance it's someone on this course. Lastly there appears to be a workaround: if you refresh the page, archive.org will load only the current page of the book. This quickly grows tiresome, however.

This bibliography will expand as time goes on. Further reading for each lecture is suggested in a slide at the end of the presentation for that week.
